# Lelmanga

Aidoku source for `https://www.lelmanga.com`.

Supports catalog/search pages, genre/status/type/order filters, manga details,
chapter lists and reader pages.

## Build

```sh
aidoku package
```

From the repository root, the full verification/build flow is:

```sh
scripts/verify-lelmanga.sh
```

## Test Locally

```sh
aidoku serve package.aix
```
